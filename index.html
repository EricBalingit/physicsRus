RigidDyn2D

RigidDyn2D is simple 2D physics engine written in Javascript. You can download source code from github.

Instruction:
1, 2, 3, 4, 5, 6, 7, 8 - select scene
r - reset scene
b - toggle bounding boxes
c - toggle contacts
j - toggle joints
s - toggle statistics
u - toggle clear rect
